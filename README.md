## Usage
You can use `GNU Stow` to create symbolic links to each file.

#### 1. Clone this repository :octocat:
First of all clone this repository in your home directory.
```sh
git clone https://github.com/50022445/dotfiles.git ~/.
```

#### 2. Create symbolic links 👽
When the repo is cloned, you can create the symbolic links with the following commands:
```sh
cd ~/dotfiles
stow .
```


That's it! 🥳
